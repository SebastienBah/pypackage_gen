# pypackage-gen

Little script to generate the general structure for a new package.

Given the name of the project, it will generate the directories and populate some basic files.

## Structure

package_name:<br/>  
┣━ README.md  <br/> 
┣━ LICENSE  <br/> 
┣━ setup.py  <br/> 
┗━ package_name:<br/> 
┃   ┗━ __init__.py<br/> 
┣━ tests:<br/> 
┃   ┗━ test_basic.py<br/> 
┗━ docs:<br/> 
    ┗━
